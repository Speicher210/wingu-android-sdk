QrCodeScannerActivity

 open class QrCodeScannerActivity : BaseSdkActivity<QrCodeScannerView, QrCodeScannerPresenter, Any>, QrCodeScannerView

This activity opens a qrcode scanner, and than calls openDetailsActivity to show content if a wingu qr code is recognized, otherwise showError will be called. The methods openDetailsActivity and showError can be overridden to customize the screen.
