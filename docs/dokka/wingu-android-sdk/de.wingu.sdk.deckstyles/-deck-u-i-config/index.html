class DeckUIConfig

This class holds UI settings that are used to render de.wingu.sdk.data.api.model.Deck

Parameters

actionColor - is used for clickable views e.g. buttons, urls

actionColorText - is text color for buttons with actionColor background, actionColorText is NOT used for urls

roundButtonCorners - if true buttons corners will be rounded

fontId - is font resource id that should be located in res/font/ folder, if null Roboto will be used e
