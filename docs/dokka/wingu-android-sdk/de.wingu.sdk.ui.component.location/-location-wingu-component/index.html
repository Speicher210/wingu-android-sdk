specMapView

 fun specMapView(): ComponentSpec

Returns specification for location component. The map is rendered using Maps Android API (com.google.android.gms.maps.MapView). This is slower than specStaticMaps, but you can restrict the usage of your Google API key to your app's package id. Google API key needs to be added to AndroidManifest.xml file, otherwise the map is not displayed. Get Google API Key.

specStaticMaps

 fun specStaticMaps(googleApiKey: StringRef? = null, urlSigningSecret: StringRef? = null): ComponentSpec

Returns specification for location component. The map is rendered using Google Static Maps API. This is faster than using specMapView, but it's not possible to restrict the usage of your Google API key. Get Google API Key.
