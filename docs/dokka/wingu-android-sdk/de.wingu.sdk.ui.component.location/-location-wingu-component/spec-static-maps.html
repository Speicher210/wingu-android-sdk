specStaticMaps

@JvmStatic fun specStaticMaps(googleApiKey: StringRef? = null, urlSigningSecret: StringRef? = null): ComponentSpec

Returns specification for location component. The map is rendered using Google Static Maps API. This is faster than using specMapView, but it's not possible to restrict the usage of your Google API key. Get Google API Key.

Parameters

googleApiKey - Optional Google API key with "Google Static Maps API" enabled (if null quota limits may apply).

urlSigningSecret - Optional URL signing secret for verifying requests to Google Static Maps API.
