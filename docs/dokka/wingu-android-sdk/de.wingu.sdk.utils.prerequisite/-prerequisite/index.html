Prerequisite

enum class Prerequisite

Holds all prerequisites required by SDK. Holds two types of location though SDK itself requires one of them.

Enum Values

ACCESS_COARSE_LOCATION

Corresponding to permission.ACCESS_COARSE_LOCATION. Required to make bluetooth scanning work.

ACCESS_FINE_LOCATION

Corresponding to permission.ACCESS_FINE_LOCATION. Required for geofencing. It is not obligatory, but if it is not met you shouldn't use geofencing feature.

BLUETOOTH_ON

Bluetooth is turned on in the device.

LOCATION_SERVICES_ON

Location turned on in the device.

PLAY_SERVICES

SDK requires that GooglePlayServices are available.
