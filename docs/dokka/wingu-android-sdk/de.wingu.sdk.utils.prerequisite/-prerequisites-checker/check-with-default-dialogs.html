checkWithDefaultDialogs

@JvmStatic fun checkWithDefaultDialogs(activity: Activity, requestCode: Int): Boolean

Checks for prerequisites. If any of them is not met starts transparent activity that will ask user to grant all necessary permissions.

Note that your activity will receive onPause() callback.

Note that it is possible that user will not grant permission for bluetooth, or location. To make sure everything has been granted you need to call .check method.

As well if there is no play services available user will be prompted to download it.

Parameters

requestCode - request code

Return
true if all prerequisites are met, false otherwise
