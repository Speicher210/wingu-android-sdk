getResolveResults

@JvmStatic fun getResolveResults(data: Intent?): ResolveResults?

This function helps check result of checkWithDefaultDialogs and getIntentIfPrerequisitesAreMissing. If any of mentioned functions returned false it means they started transparent activity for result. Using this function you can get result data form intent passed to Activity.onActivityResult.

Remember that you should verify request code passed when called previously mentioned functions.

Parameters

data - data returned by Activity.onActivityResult

Return
data object containing both resolved prerequisites and ones that failed; null if no ResolveResults object is stored in intent
