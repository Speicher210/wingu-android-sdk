NearbyChannelObserver

interface NearbyChannelObserver

Allows to receive notifications about nearby channels (beacons and geofences).

Functions

findBeaconDevice

 abstract fun findBeaconDevice(channel: Channel): IbeaconDevice?

Returns a corresponding beacon device for a nearby channel. Returns null if the channel is not nearby, or is not a beacon channel.

getChannelEventPublisher

 abstract fun getChannelEventPublisher(): Publisher<ChannelEvents>

Emits channel events. When a channel (e.g. beacon or geofence) has been discovered, updated or lost, you will receive a callback.

getChannelTimestamps

 abstract fun getChannelTimestamps(channel: Channel): ChannelTimestamps?

Returns various dates associated with the channel. E.g. a date when the channel has been seen by the user, or when a notification for the channel has been displayed.

getCurrentNearbyChannels

 abstract fun getCurrentNearbyChannels(): Collection<Channel>

Immediately returns all current nearby channels. Note that that method does not cause rescan, it returns last scan's results.

Triggers a fresh scan for nearby channels. The use of this method is optional.
