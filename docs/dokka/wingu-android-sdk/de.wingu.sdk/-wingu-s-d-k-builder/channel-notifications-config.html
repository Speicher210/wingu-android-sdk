fun channelNotificationsConfig(configFactory: Callable<ChannelNotificationsConfig>?): WinguSDKBuilder

Configures settings for background notifications. You need to enable geofenceBackgroundScan or iBeaconBackgroundScan for notifications to appear.

Parameters

configFactory - Factory for creating configuration for notifications. If null, or returns null, notifications are disabled (default).
