ibeaconBackgroundScan

@JvmOverloads fun ibeaconBackgroundScan(enabled: Boolean, scanMode: BleScanMode = BleScanMode.LOW_POWER, persistent: Boolean = false): WinguSDKBuilder

Configures if the app can scan for ibeacon channels in background mode.

Parameters

enabled - If true, the app will scan for iBeacon channel even when it's not running in foreground.

scanMode - Optional settings for scanning.

persistent - If true, the scanning process runs as a foreground service (with a notification) and is not influenced by Doze or App Standby, but may consume more battery. If false (default) the OS can delay the scanning process to optimize battery usage.
